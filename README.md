# Form Survey Perokok

Aplikasi ini merupakan aplikasi untuk pengambilan data berbasis survey terhadap seberapa banyak perokok yang dipisahkan berdasarkan umur dan gender. Untuk aplikasi ini menggunakan basis HTML, CSS, dan juga JavaScript. Setiap data yang sudah di input tidak akan hilang karena menggunakan metode local storage.
Berikut merupakan hasil dari program:

![Halaman Depan](img/Halaman%20Index.png)

![Halaman Hasil Survey](img/Halaman%20Hasil%20Survey.png)

Design dari aplikasi ini terinspirasi dari google form sehingga untuk tampilannya dibuat semirip mungkin dari google form.

## Cara menjalankan Aplikasi
1. Clone repo ini
2. Jalankan halaman index.html
3. Isi form yang disediakan dan submit
4. Secara otomatis akan langsung dibawa ke hasil survey dalam bentuk table
5. Jika ingin melakukan pengisian lagi, pencet tombol "Isi survey lagi"
